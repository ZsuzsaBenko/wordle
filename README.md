# Wordle

Wordle is a popular word game; its goal is to guess a 5-letter word in six tries at most.
Several variations exist in which players can solve more words at once (from more attempts, of course).

- The original game is available here: https://www.nytimes.com/games/wordle/index.html.
- More on its history: https://en.wikipedia.org/wiki/Wordle

This version is for Hungarian players, with Hungarian words. Enjoy!
